# 🌍 酷炫地球点名签到系统

一个功能完整、界面酷炫的课堂点名签到系统，采用3D地球旋转动画效果，支持智能概率加权点名。

## ✨ 功能特色

### 🎯 核心功能
- **📋 CSV导入名单** - 支持批量导入学生名单
- **🧠 智能概率加权** - 迟到和缺席学生点名概率自动增加
- **💾 数据持久化** - 使用IndexedDB和localStorage双重存储
- **📊 数据导出** - 支持按时间范围导出CSV记录
- **🎨 酷炫动画** - 3D地球旋转动画效果

### 📈 统计功能
- 实时统计总点名次数
- 自动计算签到率和迟到率
- 最近记录查看
- 时间范围筛选

### 🎮 交互体验
- 流畅的点名动画效果
- 直观的状态标记（签到/迟到/缺席）
- 响应式设计，支持移动设备
- 星空背景动画效果

## 🚀 快速开始

### 1. 文件结构
```
点名神器/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 核心JavaScript代码
├── 示例学生名单.csv     # 示例数据文件
└── README.md          # 说明文档
```

### 2. 运行系统
直接用浏览器打开 `index.html` 文件即可运行。

### 3. 导入学生名单
- 准备CSV文件，每行一个学生姓名
- 点击"📤 导入CSV名单"按钮
- 选择准备好的CSV文件
- 系统会显示导入成功的学生数量

## 📖 使用说明

### 名单管理
1. **导入CSV**：准备格式如下的CSV文件
   ```csv
   张三
   李四
   王五
   ```

2. **查看人数**：导入后会显示当前学生总数

### 点名操作
1. **开始点名**：点击"🌍 开始点名"按钮
2. **观看动画**：系统会展示酷炫的3D地球旋转动画
3. **显示结果**：动画结束后显示被点中的学生姓名
4. **标记状态**：选择"✅ 签到"、"⏰ 迟到"或"❌ 缺席"

### 智能加权算法
- **迟到权重**：每迟到一次，点名概率增加50%
- **缺席权重**：每缺席一次，点名概率增加30%
- **时间权重**：超过7天未被点名的学生，概率每天增加10%
- **新生权重**：从未被点名的新学生，基础概率增加50%

### 数据管理
1. **导出记录**：
   - 设置开始和结束日期
   - 点击"📥 导出记录"按钮
   - 下载CSV格式的点名记录

2. **清空记录**：
   - 点击"🗑️ 清空记录"按钮
   - 确认后删除所有历史记录

### 查看统计
- **总点名次数**：所有学生的点名次数总和
- **签到率**：签到次数 / 总点名次数 × 100%
- **迟到率**：迟到次数 / 总点名次数 × 100%
